# 🎮 Jogo da Velha  

Jogo da Velha utilizando **matriz**, desenvolvido em **JavaScript, HTML e TailwindCSS**.  
O projeto simula o clássico jogo de tabuleiro, permitindo que dois jogadores disputem partidas diretamente no navegador.
